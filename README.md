# Crypto-Volatility-tracker-forecaster-7-days-
Calculate and forecast volatility for different crypto currencies based on December 2025 data
"Daily_Aggregations.sql" File aggregates the daily high, low, open, close, volume, and return of each crypto currency at each day.
"Daily_Weekly_Volatility.sql" File calculates the volatility of each cryptocurrency for each passed day and each passed week.
"Volatility_forecast.sql" File forecasts the volatility out 7 days into the future based on December's data.
"Export_Query.sql" contains the query to export the output of "Volatility_forecast" locally.
"Volatility_forecase_proj.csv" is the exported output.
"crypto_volatility_dec2025_FINAL_2025-12-09.csv" is the initial dataset obtained from Kaggle.
